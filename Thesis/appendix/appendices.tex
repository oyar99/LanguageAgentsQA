\section{DAG and Auto DAG Agent}
\label{section:dag_prompts}

This section presents the prompts used for implementing both the DAG Agent and the Auto DAG Agent. Figure \ref{fig:prompt_base_dag} shows the base prompt used to construct the graph $G$, excluding the representative few-shot examples provided to the model during our experiments.

\input{appendix/prompt_base_dag}

\noindent Figure \ref{fig:prompt_dag_execution} shows the prompt used to execute the DAG plan in the DAG Agent. In addition, few-shot examples were provided for each supported command during our experiments. In the actual implementation, the execution loop is controlled programmatically, so the command to execute (SOLVE, ALTERNATIVE\_ANSWER, or FINAL\_ANSWER) is always known in advance. Because the agent only ever needs to respond to a single, predetermined command at each step, we removed definitions for unused commands and tools during LLM calls.
